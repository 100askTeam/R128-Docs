📄️ RTOS 存储简介

FreeRTOS 设计的完整的 IO 栈包括 iobox、vfs、fs、blkpart、flash driver。考虑到 RTOS 比 Linux 更精小高效和 Flash 不需要考虑寻址的特点，并没参考 Linux 设计 Page Cache 和 IO调度器，但在 Flash driver 添加了 Cache 以加速性能。
